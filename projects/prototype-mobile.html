Prototype d'Application Mobile

Conception d'une application mobile intuitive pour la gestion de tâches, centrée sur l'expérience utilisateur et l'accessibilité.

Figma UX Research UI Design Prototypage User Testing

Vue d'Ensemble

Ce projet académique avait pour objectif de créer une application mobile moderne pour la gestion de tâches et de projets personnels. L'accent a été mis sur une approche centrée utilisateur, avec une interface intuitive et une expérience fluide sur mobile.

Le défi principal était de simplifier un système complexe de gestion de tâches tout en conservant des fonctionnalités avancées pour les utilisateurs expérimentés. L'application devait être accessible aux débutants mais suffisamment puissante pour les power users.

Fonctionnalités Principales

📋

Gestion de Tâches

Création, édition et organisation de tâches avec système de priorités, dates d'échéance et catégories personnalisables.

🔄

Synchronisation Cloud

Synchronisation automatique entre appareils avec sauvegarde sécurisée et accès hors ligne complet.

Interface Personnalisable

Thèmes sombres/clairs, couleurs d'accent personnalisables et organisation flexible des vues.
